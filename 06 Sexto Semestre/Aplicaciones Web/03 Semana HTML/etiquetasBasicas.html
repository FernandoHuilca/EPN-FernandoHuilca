Párrafos

La etiqueta < p > se utiliza para definir párrafos en HTML.

Los párrafos son bloques de texto que se separan por un espacio en blanco.

Enlaces

La etiqueta < a > se utiliza para crear enlaces en HTML.

Los enlaces permiten a los usuarios navegar entre diferentes páginas web o secciones dentro de una misma página.
